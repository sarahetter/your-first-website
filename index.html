What is HTML?

  • Stands for HyperText Markup Language
  • Language that a browser understands to display webpages
  • Consists of different tags

HTML Tag

  • A tag describes its contents
  • Tags come in pairs. They have a opening tag, and a closing tag
  • <tag> Whatever the tag describes </tag>

What is CSS?

  • Stands for Cascading Style Sheets
  • It is where we add style to our webpage

CSS Syntax

h1 {
    color: purple;
    font-size: 32px;
}

Adding Stylesheets

  • Add the following blue code in the <head> section, then go and check your website.
<head>
    
    Your Website Title Here
    <link rel="stylesheet" href="css/normalize.css">
    <link rel="stylesheet" href="css/mystyle.css">
</head>

Colours

  • You can change colours with css
  • In css, colour is spelled the American way, "color"
body {
    background-color: orchid;
}
header {
    color: steelblue;
    background-color: lightsalmon;
}

Fonts

  • You can customize the text to your liking
  • You can change the size, font, boldness, italics, etc.
h1 {
    font-size: 50px;
    font-family: 'Pacifico';
}
h2 {
    font-family: 'Chewy';
}
